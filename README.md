# EchoNova - Software Solutions Website

موقع ويب احترافي لعرض خدمات الشركات التقنية وأعمالها.

## المحتويات

الموقع يتكون من الصفحات التالية:

- الصفحة الرئيسية مع عنوان جذاب
- قسم الخدمات التي تقدمها الشركة
- معرض الأعمال السابقة
- نموذج التواصل مع العملاء

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- Montserrat Alternates Font

## المميزات

- تصميم متجاوب يعمل على جميع الأجهزة
- نظام بحث مباشر في المحتوى
- قائمة تنقل تتفاعل مع السكرول
- نموذج اتصال مع التحقق من المدخلات
- رسائل خطأ واضحة للمستخدم
- تأثيرات حركية بسيطة وسلسة
- معرض صور للمشاريع السابقة

## البنية

```
tech/
├── index.html          - الصفحة الرئيسية
├── styles.css          - ملف التنسيقات
├── script.js           - ملف البرمجة
└── assets/             - مجلد الصور
    ├── 1.jpg
    ├── 2.jpg
    ├── 3.jpg
    └── ...
```

## طريقة الاستخدام

1. افتح ملف index.html في المتصفح مباشرة
2. أو استخدم خادم محلي لتشغيل الموقع

## الأقسام الرئيسية

### قسم البطل (Hero)
يحتوي على عنوان رئيسي ووصف للشركة مع زر للانتقال لنموذج التواصل.

### قسم الخدمات
يعرض ثلاث خدمات أساسية:
- تطوير المواقع
- البرمجيات المخصصة
- الاستشارات التقنية

### قسم الأعمال
معرض صور يعرض المشاريع السابقة مع تأثيرات تفاعلية.

### قسم التواصل
نموذج بريد إلكتروني مع:
- حقل الاسم
- حقل البريد الإلكتروني
- حقل الرسالة
- التحقق من صحة المدخلات

## ملاحظات

- الموقع جاهز للاستخدام المباشر
- يمكن تخصيص الألوان من خلال CSS Variables
- نموذج الاتصال يعمل بشكل تجريبي ويحتاج لربطه بخادم حقيقي
